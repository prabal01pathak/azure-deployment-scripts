# deployment-scripts

> azure deployment scripts

## Install

Install with [poetry](https://www.poetry.com/)

```sh
$ poetry shell
$ poetry install
```

## Usage

```sh
$ python run.py
```

## Running tests

Install dev dependencies:

```sh
$ poetry run tests
```
